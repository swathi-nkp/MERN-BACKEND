Learning JavaScript(Js)

Introduction

What is JavaScript??

JavaScript is a high-level programming language mainly used to make websites interactive and dynamic. It runs in web browsers like Google Chrome, Mozilla Firefox, and Microsoft Edge, and it can also run on servers using environments like Node.js.

Here’s a simple JavaScript example:

Alert("Hello, world!");

    • Topics

    Variables

    Variable is a named container that stores data, like numbers, text, objects, or other values, which you can use and change in your code.

      Ways to Declare Variables
  • Var - old-style, global if declared outside a function.
  • Let - modern, preferred for most variables.
  • Const - for constant values that shouldn’t change.

    • Data types

    JavaScript has 8 main data types, divided into Primitive and Non-Primitive (Reference) types.

      1.Primitive Data Types
  • String: ""(or)''
  • Number: 123...
  • Boolean: True (or) False
  • Undefined:Not assigned value
  • Null: Empty value
  • BigInt: 123456789012345678901234567890n
  • Symbol: Value is unique
      • 2.Non-Primitive Data Types
  • Object: { }
  • Arrays: [ ]
  • Functions: (){ }

    • Operators

    Arithmetic Operators: ( + , - , * , / , % , ++ , -- )

    Assignment operators: ( = , += , -= , *= , /= , %= , **= )

    Comparison Operators: ( ==, ===, !=, !==, >,<, >=, <= )

    Logical Operators: ( && (AND), || (OR), ! (NOT) )

    Bitwise Operators: ( &, |, ^, ~, <<, >>, >>> )

    Ternary Operator: (condition ? valueIfTrue : valueIfFalse)

    Type Operators: (typeof, instanceof)


    Scopes

    Global Scope: Access everywhere in field

    Function Scope: Access only that block

    Block Scope: Access only inside that function

    Lexical Scope: Declared values can call only like nested.
